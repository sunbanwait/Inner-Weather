# InnerWeather 🌤️

InnerWeather is a Chrome Extension designed to help users track and visualize their daily moods directly from their browser. By logging emotional states over time, it provides a simple, intuitive calendar view of your "internal weather."

## Features!

* **Daily Mood Logging:** Quickly record how you're feeling without interrupting your workflow.
* **Mood Calendar (`MoodCalendar.jsx`):** A visual grid tracking your emotional patterns over days, weeks, and months.
* **Customizable Configurations (`moodConfig.js`):** Easily adjustable settings for different mood types, colors, and tracking metrics.
* **Seamless Browser Integration:** Lightweight extension that lives right in your Chrome toolbar.

## Stack!

* **Frontend Framework:** [React](https://reactjs.org/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Styling:** CSS (`App.css`, `index.css`)
* **Environment:** Chrome Extensions API V3
